# 🏨 Hotel Reservation System

A robust backend application built to manage hotel bookings, room availability, and customer records. This project demonstrates core Object-Oriented Programming (OOP) principles and database connectivity in Java.

## 🚀 Features

* **View Available Rooms:** Dynamically checks and displays rooms that are currently open for booking.
* **Make a Reservation:** Collects guest details and secures a room booking in the system.
* **Update/Cancel Bookings:** Allows modification or deletion of existing reservations.
* **Database Integration:** Uses JDBC to persistently store guest data, room status, and transaction history.

## 🛠️ Technologies Used

* **Language:** Java
* **Architecture:** Object-Oriented Programming (OOP)
* **Database:** MySQL (via JDBC Driver) 
* **Environment:** Eclipse IDE
